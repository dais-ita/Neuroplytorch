BasicLSTM

class training.basic_models.BasicLSTM(input_size: int = 10, output_size: int = 4, **kwargs)[source]

Pure Pytorch implementation of a very simple LSTM model with a single layer followed by a dense classification layer

__init__(input_size: int = 10, output_size: int = 4, **kwargs)[source]
Parameters

  • input_size (int, optional) – Size of the input data (each element of the sequence). Defaults to 10.

  • output_size (int, optional) – Size of the output, for the case of Neuroplex this would be the number of complex events, resulting in an output vector of size output_size. Defaults to 4.

  • hidden_size (int, optional) – LSTM hidden layer size. Defaults to 64.

  • use_relu (bool, optional) – Use ReLU on output or not. This is for use with Evidential Deep Learning loss functions. Defaults to False.

forward(x: None._VariableFunctionsClass.tensor) → None._VariableFunctionsClass.tensor[source]
